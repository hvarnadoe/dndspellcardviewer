# D&D 5e Spell Card Viewer (2024 Edition)

A powerful, cross-platform tool built with **Avalonia UI** and **.NET 8** designed to browse, filter, and export the latest 2024 Dungeons & Dragons spells into printable PDF spell cards.

**[Try the Web Version](https://hvarnadoe.github.io/dndspellcardviewer/)**

---

## Key Features

* **2024 Core Ruleset**: Specifically curated for the 2024 5e spell updates.
* **PDF Export**: Generate high-quality spell cards ready for your tabletop sessions.
* **Cross-Platform**: Native desktop applications for **Windows** and **macOS** (Intel & Apple Silicon).
* **Instant Search**: Filter by spell name, class, or level to find exactly what you need.

---

## Installation

### Desktop (Recommended)
The desktop versions offer the smoothest experience for generating and saving PDFs.
1.  Go to the **[Releases](https://github.com/hvarnadoe/dndspellcardviewer/releases)** tab.
2.  Download the version compatible with your OS:
    * **Windows**: `SpellViewer.exe`
    * **macOS (Apple Silicon)**: `SpellViewer_AppleSilicon` (for M1/M2/M3 Macs)
    * **macOS (Intel)**: `SpellViewer_Intel`
3.  Run the executable and start preparing your spellbook!

### Web
Access the viewer directly in your browser at the link above. Note that web browsers may handle PDF downloads differently than the desktop app.
